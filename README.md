# LifeTrack — Your life. Your data. Your truth.

Free, local, privacy-first life analytics. Tracks every app and site on your laptop and generates a brutal honest daily report.

## Setup
```bash
pip install -r requirements.txt
set GROQ_API_KEY=your_key   # get free key at console.groq.com
python main.py
```

## Commands
```bash
python main.py              # start tracking
python main.py report       # today's report
python main.py week         # 7-day summary
```
